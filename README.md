Зарегистрироваться на GetBlock.io.
Используя API GetBlock создать сервис, выводящий адрес, баланс которого изменился (в любую сторону) больше остальных за последние сто блоков.

Получить номер последнего блока можно с помощью следующего метода:
https://getblock.io/docs/eth/json-rpc/eth_eth_blocknumber/


А данные блока вместе с транзакциями через
https://getblock.io/docs/eth/json-rpc/eth_eth_getblockbynumber/

Важно: API ключи в репозитории не хранить.
Допускается использовать только стандартные библиотеки Golang(https://pkg.go.dev/std)
